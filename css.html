CSS(cascading style sheet)

what is css?

  • css stands for cascading style sheet
  • css describe how HTML element are to be displayed on screen,paper,or in other media
  • It can control the layout of multiple web pages all at once

CSS syntax

A css rule-set consist of a selector and a declaration black:

h1 {color(property):blue(value); font-size:12px;}

CSS selector

1. Type selector

  • A type selector is a name of HTML tag
    • p{text-align:center;color:red;}
  • the rule applies to all <p> tags in document

2.ID selector

  • The id selector uses the id attribute of an html element to select a specific element
  • To select an element with speific id, write a hash(#) character, followed by the id of the element
    • #para1{text-align:center;color:red;}
  • An id name start with a number!

    3.class selector

  • The class selector select html element with a specific class attribute.
  • to select element with a specific class, write a period (.) characte , followed by the class name.
    • .center {text-align:center; color:red;}

    4.Grouping

  • The grouping selector select all the HTML element with same style definations
    • h1, h2, p {text-align:center; color:red;}

    5.Universal selector

  • the universal selector (*) select all the HTML elements on the page.
    • *{text-align:center; colr:red;}

pepeat Default. The background image will be replicated horizontally and vertically

No-repeat The backgroung-image will be displayed only once

  • The background-position property is used to specify the position of the background-image.

  • The background-attachment property specifies wether the background image should scroll or to be fixed

Padding-box (default) the background image start from the upper left corner of the padding edge

Position property

    The position property specifies the type of positioning method used for an element

  • static
  • Relative
  • Fixed
  • Absolute
  • Sticky

    1.Position:static;

  • static positioned element are not affected by the top,bottom,left, and right properties
    • div.static{position:static; border:3px solid #73AD21;}

    2.Position:relative;

  • An element with position:relative; is positioned relative to its normal position
    • div.Relative{position:relative; border:3px solid #73AD21;}

    3.Position:Fixed;

  • An element with position:fixed; is positioned relative to viewport, which means it always stay in the same place even if the page is scrolled.
    • div.fixwd{position:fixed; bottom:0; width:300px; right:0; border:3px solid #73AD21;}

    4.Position:absolute;

  • An element with position:absolute; is positioned relative to the nearest positional ancestor
    • div.absolute{position:absolute; top:80px; height:100px; bottom:0; width:300px; right:0; border:3px solid #73AD21;}

    5.position:sticky;

  • An element with position:sticky; is positioned based on user scroll position.
    • div.sticky{position:-webkit-sticky; /*safari*/ position:sticky; top:0;n background-color:green; border:2px solid #4CAF50}

    6.overlapping element

  • The z-index property specifies the stack order of an element (which element should be placed in front of, or behind,the others).
    • img{position:absolute; left:0; top:0; z-index:-1;}

    CSS Transform

  • 1. 2D Transforms
  • CSS transform allow you to move,rotate,scale, and skew elements.
  • The translate() method
  • The rotate() method rotates an element clockwise or counter-clockwise according to a certain degree.
  • Example:- div{transform:rotate(20deg);}
  • The rotate() method
  • The translate method() moves an element from its current position(according to the parameters given for the X-axis and the Y-axis).
  • Example:- div{transform:translate(50px, 100px);}
  • The scale() Method
  • The scale() method increase or decrease the size of an element(according to the parameter given for the width and the height).
  • Example:- div{transform:scale(2,3);}
  • The scaleX() method:- The scaleX() method increase or decrease the width of an element.Example: div{transform:scaleX(2)}
  • The scaleY() method:- The scaleY() method increase or decrease the height of an element. Example:div{transform: scaleY(3);}
  • The skew method:- The skew() method skews an element along the X and Y-axis given by the angles
  • Example:-div{transform:skew(20deg, 10deg);}
  • The skew() method skews an element along the X and Y-axis given by angles.Example:-div{transform:skew(20deg, 10deg);}
  • The matrix() method:-The matrix() method take six parameters, contaning mathematic functions, which allows you to rotate, scale, move(translate), and skew elements.

    • The parameters are as follow: matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY()) div{transform:matrix(1,-0.3,0,1,0,0);}

    3 3D Transforms

  • With the CSS transform property you can use the following 3d transformation methods:
  • The rotateX() method:- The rotate X() method rotates an element around it's X-axis at a given degree: Example:-#myDiv{transform:rotateX(150deg);}
  • The rotateY() method:- The rotateY() method rotates an element around it's Y-axis at a given degree: Example:-#myDiv{transform:rotateY(150deg);}
  • The rotateZ() method:- The rotateZ() method rotates an element around it's Z-axis at a given degree: Example:-#myDiv{transform:rotateZ(150deg);}

CSS(cascading style sheet)

what is css?

  • css stands for cascading style sheet
  • css describe how HTML element are to be displayed on screen,paper,or in other media
  • It can control the layout of multiple web pages all at once

CSS syntax

A css rule-set consist of a selector and a declaration black:

h1 {color(property):blue(value); font-size:12px;}

CSS selector

1. Type selector

  • A type selector is a name of HTML tag
    • p{text-align:center;color:red;}
  • the rule applies to all <p> tags in document

2.ID selector

  • The id selector uses the id attribute of an html element to select a specific element
  • To select an element with speific id, write a hash(#) character, followed by the id of the element
    • #para1{text-align:center;color:red;}
  • An id name start with a number!

    3.class selector

  • The class selector select html element with a specific class attribute.
  • to select element with a specific class, write a period (.) characte , followed by the class name.
    • .center {text-align:center; color:red;}

    4.Grouping

  • The grouping selector select all the HTML element with same style definations
    • h1, h2, p {text-align:center; color:red;}

    5.Universal selector

  • the universal selector (*) select all the HTML elements on the page.
    • *{text-align:center; colr:red;}

  • An internal style sheet may be used if one single HTML page has a unique style.
  • You can put style sheet rules in the head of the document by <style >
    • <head >

    body{background-color:linen;}
    h1{color:maroon; margin-left:40px;}
    < /style>
    < /head >

3.Inline CSS

  • An inline style may be used to apply a unique style for a singl element
  • The start tag can contain style sheet rules directly in HTML documents by the style attribute
    • <body>
    <h1 style="color:blue; text-align:center;">This is a heading </h1> <p style="color:red;">This is a paragraph.</p> </body>

    CSS comments

  • comments are used to explain the code,and may help when you edit the code on the later date.
  • A CSS comment is placed inside the element, and start with / * and ends with */:
    • <style> / * this is a single-line comment */p {color:red;} </style>

Background properties

    1.background-color

  • The background-color property specifies the background color of an element
    • h1{background-color:green;}
  • opacity/transparency:- The opacity property specifies the opacity/transparency of an element.it can take a value from 0.0-1.0
    • p{ background-color:green; opacity:0.3;}

    2.background-image

  • The background-image property specifies an image to use as the background of the element
    • Example: body{background-image:url("paper.gif"):}

    3.background-repeat

  • sets how a background image will be repeated
Value Description
pepeat Default. The background image will be replicated horizontally and vertically
Repeat-x The background image will be repeated horizontally
Repeat-y The background will be repeated vertically
No-repeat The backgroung-image will be displayed only once

body{background-image:url("gradient_bg.png"); background-repeat:repeat-x;}

    4.background-position

  • The background-position property is used to specify the position of the background-image.
    • body{ background-image:url("img_tree.png"); background-repeat:no-repeat; background-position:right-top; }

    5.background-attachment

  • The background-attachment property specifies wether the background image should scroll or to be fixed
    • body{ background-image:url("img_tree.png"); background-repeat:no-repeat; background-position:right-top; background-attachment:fixed; }

    6.background-size

  • specifies the size of the background images.
    • body{ background-image:url("img_tree.png"); background-repeat:no-repeat; background-position:right-top; background-attachment:fixed; background-size:80px 60px; }
  • Contain: scale the image to the largest size such that both its width and its height can fit inside the content area
  • Cover: Scale the background image to be as large as possible so that the background area is completely covered by the background image

    7.background-origin

  • The CSS background-origin specify where the background image is positioned.
value Description
border-box the background image startsfrom the upper left corner of the border
Padding-box (default) the background image start from the upper left corner of the padding edge
content-box the background image start from the upper left corner of the content

    background-clip

  • The CSS background-clip property specifies the painting area of the background.
    • Value Description
    border-box (default) the background is painted to the outside edge of the padding
    content-box the background is painted within the content box

    background-Shorthand property

  • Specifies all the background properties in one single property
  • When using the shorthand property the order of the property value is:
  • background-color
  • background-image
  • background-rtepeat
  • background-attachment
  • background-position
    • body{ background:#ffffff url("img_tree.png")no-repeat right-top; }

Text properties

    1.Text color

  • The color property is used to set the color of the text.The color is specified by:
    • {color:blue;} h1{color:green;}

    2.Text-allignment

  • The text-align property is used to set the horizontal allignment of a text
  • A text can be left or right aligned,centered,or justified
    • h1{text-align:center;}

    3.Vertical-allignment

  • the vertical-allign property set the vertical allignment of the element
    • img.top{vertical-align:top;} img.middle{vertical-align:middle;} img.top{vertical-align:top;}

    4.Text-decoration

  • The text-decoration property is used to set or remove decoration from the text
  • The value text-decoration:none; is often used to remove underline from the links:
    • Value Description
    None Default.Defines a normal text
    Underline Defines a line under the text
    overline Defines a line over the text

    5.Text transformation

  • The text-transformation property is used to specify the uppercase and lowercase letters in the text
    • p.uppercase{text-transform:uppercase;} p.lowercase{text-transform:lowercase;}

    6.Text-indentation

  • The text-indent property is used to specify the indentation of the first line of the text.
    • Example:- p{text-indent:50px;}

    7.Letter spacing

  • The letter-spacing property is used to specift the space between characters in a text
    • Example:-h1{letter-spacing:3px;}

    8.Line Height

  • The line height property is used to specify the space between the lines:
    • p.small{line-height:0.8;} p.big{line-height:1.8;}

    Word spacing

  • The word spacing property is used to specify the space between the word in a text
    • h1{word-spacing:10px;}

    10.White space

  • The white space property specifies how white space inside an element an is handeled
    • p{white-space:nowrap;}
    Value Description
    Normal Default.White space is ignored by the browser
    pre White-space is preserved by the browser.Acts like the <pre> tag in HTML
    Nowrap The text will never wrap, it continues on the same line until a <br> tag is used

    11.Text-shadow

  • The text shadow property add shadow to the text
    • h1{text-shadow:2px 2px;}
  • In its simplest use, you can only specify the horizontal shadow(2px) and the vertical shadow(2px):
  • Text shadow color
    • h1{text-shadow:2px 2px red;}
  • Blur effect to shadow
    • h1{text-shadow:2px 2px 5px red;}

    12.Text overflow

  • Specifies how overflowed content that is not displayed should be signled to the user
  • clipped:- clip the text
  • Ellipsis:-Render an ellipsis(...) to represent clipped text

    13.Word wrapping

  • The CSS word-wrap property allow long word to be able to broken and wrap onto the next line.
    • <style> p.test{width:1em; border:1px solid #000000; word-wrap:break-word;} ></style>

    14.Word-breaking

  • The CSS word-break property specifies line breaking rules.
  • Keep-all
  • Break-all
    • p.text1{word-break:Keep-all;}

    15.Writing mode

  • Specifies whether the line of text are laid out horizontally or vertically.
    • p.test1{writing-mode:horizontal-tb;} span.test2{writing-mode:vertical-rl;}

    1.Font Family

  • Font-family property is a prioritized list of font family name for an element
    • .serif{font-family:"Times New Roman",Times,serif;}

    2.Font style

  • set the font style
    • Value Description
    Normal Default.Display a normal font
    Italic Display an italic font
    Oblique Display an Oblique font
    p.normal{font-style:normal;} p.italic{font-style:italic;}

    3.Font-weight

  • Specifies the weight of a font
    • Value Description
    Normal Default.Define normal characters
    Bold/Bolder Defines thick/thicker characters
    Lighter Defines lighter character
    100-900(with the intertval of 100) Defines from thin to thick characters 400 is the same as normal, and 700 is the same as bold
    p.thick{font-weight:bold;}

    4.Font variant

  • Specifies whether or not a text should be displayed in a small-caps font
  • In a small-caps font, all lowercase letter are converted to uppercase letter. however the converted uppercase letter appears in a smaller font size than the original uppercase letter in the text.
    • p.small{ font-variant:small-caps; }

    5.font-size

  • Sets the size of the text
  • Font size with the pixels
    • h1{ font-size:40px; }
  • Font size with em:- 1em is equal to the current font sizw. The default text size in browsers is 16px.
    • h1{font-size:2.5em; /*40px/16-2.5em*/}

    6.Font-stretch

  • the font stretch property allow you to make text narrower(condensed) or wider(expanded).

    • Responsive font size

  • the text sizer can be set with a VW unit, which means the "Viewport width"
  • That way the text size will follow the size of the browser window.

    • Font shorthand

  • to shorthand the code, it is also possible to specify all the individual font properties in one property
  • font-style
  • font-variant
  • font-weight
  • font-size
  • font-family

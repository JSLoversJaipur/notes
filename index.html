Tags and Attributes

Attribute

  • Attributes provide extra information about a tag. For instance if we want to place an image on website.
    we must tell our browser where the image is located.
  • Many HTML tags have a unique set of their own attributes. For example if you need to set background color of body, you must use bgcolor attributes of the body tag.
  • Example

    • <body bgcolor="black>My text here.</body>

Title attribute

  • This attributes title an element and adds a tiny text pop-up to any HTML element much like the text
    pop-ups of word processing programs.
  • <h2 title="Hello There!">Titled Heading Tag</h2>
  • Rollover your mouse over the heading to see the effect of this attribute!

Align Attribute

  • If you wish to change the horizontal location of your elements you may do using the align
    attribute. you may align things left, right or center.
  • Example-

    • <h2 align="left">Left align heading </h2>
    <h2 align="center">Centered Heading </h2>
    <h2 align= "right">Right aligned heading </h2>

Note-Attribute Defaults

  • Many tags are assigned defaults.
  • For example, a pargraph tag will always align its text to the left unless it has an align attributein it specifying
    Otherwise.

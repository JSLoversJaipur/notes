Other Tags

  • <br> - Break tag for line break.
  • <hr> - The <hr> tag define a thematic break in an Html page.
  • <pre> - The HTML <pre> element defines preformated text.
  • <img> - To insert image in document.

HTML Headings

  • HTML headings are titles or subtitles that you want to display on the webpage.
  • HTML headings are defined with <h1> to <h6> tags
  • <h1> defines the most important heading <h6> defines the least impotant heading.

HTML Paragraph

  • The HTML <p> elements defines a paragraph.
  • A paragraph always starts on a new line,and browser automatically add some white space
    before and after a paragraph.

Example

  • open text editor(Notepad)
  • Type following example <html>
    <body>
    <h1>My first heading</h1>
    <h2>My second heading</h2>
    <hr>
    <p>My First pargraph</p>
    </body>
    </html>
  • Save file as myfirstpage.html
  • Run it on your browser.(Chrome,Mozilla etc)

Fonts and Formating

  • The <font> tag is used to add style, size,and color to the text on your site, Use the size, color and face.
  • Use a <basefont> tag to set all of your text to the same size,face, and color.

Font Size

  • Set the size of your font.
  • Example
    <p>
    <font size="5">Here is a size 5</font>
    </p>

    HTML <picture> tag

    • The HTML <picture> element gives web developers more flexibilities in specifying image resources.
    • The <picture> element contains One or more <source> elements, each referring to different images
      through the srcset attribute. This way the browser can choose the image that best fits the current view
      and/or devices.
    Example
    <picture>
    <source srcset="img_food.jpg">
    <source srcset="img_car.jpg">
    <source srcset="img_girl.jpg">
    </picture>

    When to use the Picture Element

    • Bandwidth:- If you have a small screen or device, it is not necessary to load a large image file. The
      browser will use the the first <source> element with matching attribute values, and ignore any of the following elements.
    • Format Support:-Some browser or devices may not support all image formats will use the first formats. By using the <picture> element, you can add images of all formats, and the browser will use the first format it
      recnore any of the following elemognizes, and ignore any of the following elements.

    What are Forms

    • A form is an area where users are allowed to enter information such as their name phone no, email, address, query etc
    • A form is defined with the <form> tag.

    Form tag Attributes

    1. Action:- Specifies the name of the file to send the content to
